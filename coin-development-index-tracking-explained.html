    • Coin Ranking

    We loosely follow cryptocoin market cap rankings and add new cryptocoins in that order. In the future we aim to provide a weighted developement index ranking of all the cryptocoins available on the cryptocurrency markets.

    Code Library

    We track the core code repository of each cryptocoin. We don't see any value in tracking code repositories that house for example, the crypto wallet or their website. The link provided in the development index will take you directly to the core code repository.

    Development Team

    The cryptocoin development team size is one of the most important drivers of future success. The number of developers found in the development index represent front-end, back-end, blockchain and security developers on their respective cryptocoin teams. We track the number of developers that actively contribute to the code base on their teams. We actively reach out to crytpocoins teams to confirm developer team sizes.

    Developer Documentation

    A cryptocoin team provides developer documentation to drive engagement with their developer community. The base expectation of documentation is that it accurately describes what's possible with an API or developer tool. A quickstart or getting started guide plays an important role in introducing new technology to developers.

    Repositories

    If the cryptocurrency team has a code library on Github, Bitbucket or Gitlab. Their code will be stored in repositories. We currently track updates to the core code repository only, however, we track the total number of repositories associated with their code library.

    Stars

    If a repository is popular with developers they will star the repository. If your cryptocurrency team has high starred repositories it signals a lot of developers are watching this repository and possibly interacting with the repository. The number of stars in the development index represent the total amount of stars associated with all the repositories in their Github, Gitlab and Bitbucket account.

    Developer Contributors

    A developer contributor is someone who is not part of the cryptocoin team, however, they are able to contribute to the public respositories that the team has created. A high number of developer contributors is a good sign that the cryptocurrency software is well received by the developer community.

    Latest Update

    We only track the latest updates to the core code repository. We do not track updates to other repositories in their respective Github, Gitlab and Bitbucket accounts.

    Current Release

    You will see the "In Progress" label attached to any cryptocoin account where they are actively developing their next update. Cryptocoin teams that have stopped developing their software will have a "Stopped" label attached to their account.

    Top Languages

    We list the top programming languages associated with their cryptocurrency software platform.
